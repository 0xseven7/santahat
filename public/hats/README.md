# 圣诞帽图片说明

请将您的圣诞帽图片放在此目录下，并按照以下命名规则：

- `hat-red.png` - 红色圣诞帽
- `hat-yellow.png` - 黄色圣诞帽
- `hat-green.png` - 绿色圣诞帽
- `hat-blue.png` - 蓝色圣诞帽
- `hat-purple.png` - 紫色圣诞帽

## 图片要求

- 格式：PNG（支持透明背景）
- 建议尺寸：正方形或接近正方形
- 背景：透明背景效果最佳
- 圣诞帽应该朝下，以便正确叠加在头像上

## 示例

如果您想添加更多颜色，只需：
1. 将图片放在此目录
2. 按照 `hat-{颜色名}.png` 的格式命名
3. 在 `components/ColorSelector.tsx` 中添加对应的颜色选项
